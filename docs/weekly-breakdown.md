# Weekly Breakdown

This course is structured over 13 weeks, divided into 4 core modules.

## Module 1: The Nervous System (Weeks 1-3)
**Focus**: ROS 2, Linux, and Python Middleware.

*   **Week 1**: Introduction to Linux, File Systems, and Bash Scripting.
*   **Week 2**: ROS 2 Fundamentals (Nodes, Topics, Services).
*   **Week 3**: Creating Custom Messages and Python Middleware.

## Module 2: The Digital Twin (Weeks 4-6)
**Focus**: Simulation and URDF Modeling.

*   **Week 4**: Introduction to URDF and Mesh formats (STL/DAE).
*   **Week 5**: Physics Engines (MuJoCo/Isaac Sim) and Environment Setup.
*   **Week 6**: Sim-to-Real Transfer strategies and Kinematics.

## Module 3: The AI Brain (Weeks 7-9)
**Focus**: Imitation Learning and Teleoperation.

*   **Week 7**: Teleoperation Hardware (ALOHA) and Data Collection.
*   **Week 8**: Training Policies (ACT, Diffusion).
*   **Week 9**: Evaluation and Tuning of Policies.

## Module 4: VLA & Deployment (Weeks 10-13)
**Focus**: Vision-Language-Action Models and Edge Deployment.

*   **Week 10**: Introduction to VLMs and Transformers in Robotics.
*   **Week 11**: Integrating Foundation Models for Planning.
*   **Week 12**: Deployment on Jetson Orin and Optimization.
*   **Week 13**: Capstone Project Showcase and Demo Day.
