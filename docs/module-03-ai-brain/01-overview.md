# Module 3: The AI Brain

## Focus
Teaching robots to perform tasks through demonstration and imitation learning.

## Key Topics
- **Teleoperation**: ALOHA and SoR (Stick-on-Robot) data collection.
- **ACT**: Action Chunking with Transformers.
- **Diffusion Policy**: Handling multimodal distributions.
- **Training Pipeline**: Data processing, training, and evaluation.
