## Setup

1.  **Install Dependencies**: `npm install`
2.  **Start Dev Server**: `npm run start`

## Verification Checklist

### Visual Style
- [ ] **Headings**: Verify all headings (H1-H6) are using the serif font (Merriweather).
- [ ] **Body Text**: Verify body text is sans-serif.
- [ ] **Whitespace**: Confirm the layout feels "open" and not cluttered.

### Content Sections
- [ ] **Hero**: Check for Title ("Physical AI & Humanoid Robotics"), Subtitle, and "Get Started" button.
- [ ] **Features Grid**: Verify 3 columns highlighting ROS 2, Digital Twins, and VLA.
- [ ] **Curriculum**: Verify the list of modules is present and links work.

### Responsiveness
- [ ] **Mobile**: Resize browser to <768px. Ensure the grid stacks vertically.
