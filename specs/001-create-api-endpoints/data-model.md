# Data Model: Create API Endpoints for Frontend

**Feature**: Create API Endpoints for Frontend

## Entities

### UserQuery

Represents the input from the frontend user.

| Field | Type | Required | Description | Constraints |
|-------|------|----------|-------------|-------------|
| user | string | Yes | The natural language query from the user. | Non-empty string. |

### AgentResponse

Represents the response from the backend agent.

| Field | Type | Required | Description | Constraints |
|-------|------|----------|-------------|-------------|
| assistant | string | Yes | The natural language response generated by the agent. | - |

## Pydantic Models (Reference)

```python
from pydantic import BaseModel, Field

class UserQuery(BaseModel):
    user: str = Field(..., description="The user's query", min_length=1)

class AgentResponse(BaseModel):
    assistant: str = Field(..., description="The agent's response")
```
