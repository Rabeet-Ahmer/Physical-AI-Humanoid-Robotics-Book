# Quickstart: Implementing Module 1

**Branch**: `002-ros2-nervous-system`

## Prerequisites
1.  Node.js 18+ installed.
2.  `npm install` run in project root.
3.  Branch `002-ros2-nervous-system` checked out.

## Implementation Steps

1.  **Create Directory**: Ensure `docs/module-01-nervous-system/` exists.
2.  **Create Files**: Generate the 9 Markdown files defined in `data-model.md`.
3.  **Populate Content**:
    -   Use `01-overview.md` for the "Concept Overview" and "System-Level Intuition".
    -   Use `03-theory-fundamentals.md` for Math/Models.
    -   Insert Mermaid diagrams where required.
4.  **Verify**:
    -   Run `npm start` to preview.
    -   Check `http://localhost:3000/docs/module-01-nervous-system/overview`.

## Testing
-   **Build Check**: Run `npm run build` to ensure no broken links or MDX errors.
-   **Visual Check**: Verify diagrams render and math equations ($$) appear correctly.
